# 세션 종료 프롬프트

이 파일의 프롬프트를 세션 종료 시 사용자 입력으로 사용한다.

## 사용
- 레벨: L0-도구 (세션 종료 시 사용)  
- 읽기: 세션 종료 시.  
- 갱신: 종료 절차/요약 규칙이 변경되었을 때.  
## PROMPT
```
세션 종료. 아래 절차를 반드시 수행하라.
1) docs/session/README.md 업데이트: 프로젝트 스냅샷/현재 상태를 최신화하고, 없으면 "없음"으로 명시.
2) docs/session/LOG.md에 새 항목 추가(최신 상단). 날짜/시간, 목표/변경/파일/참조/결정/테스트/다음 항목을 채운다. 참조는 세션 중 읽은 모든 문서(L0 포함).
3) 이번 세션에 영향이 있는 문서(L1/L2/L3)를 갱신한다. 변경이 없으면 "변경 없음"으로 기록.
4) 응답에는 변경된 파일 목록과 테스트 실행 여부를 요약한다.
```
