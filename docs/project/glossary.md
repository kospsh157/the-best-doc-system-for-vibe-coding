# 도메인 용어

## TL;DR (10줄 이내)
- 핵심 용어만 간단히 정의한다.  
- 중복 정의는 금지한다.  

## 사용
- 레벨: L1 (작업 재개/용어 확인 시)  
- 읽기: 새로운 도메인 용어가 등장하거나 혼동이 있을 때.  
- 갱신: 용어 정의가 추가/변경되었을 때.  

## 용어
- 세션 복귀: 새 세션 시작 시 문서만으로 이전 작업 맥락을 빠르게 복원하는 과정
- L0: 세션마다 반드시 읽는 핵심 문서 레벨
- L1: 작업 재개 판단에 필요한 보조 문서 레벨
- L2: 특정 작업 영역에서만 읽는 상세 문서 레벨
- L3: 심화 참조/장문 문서 레벨
- 재개 포인트: 다음 세션에서 바로 시작할 1개 작업을 명시한 문서 항목
- 구조 강제 장치: 계층 경계를 자동 검증하는 `scripts/check_arch.py`
